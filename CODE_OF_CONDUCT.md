# Code of Conduct

This project is for developers who want practical, reproducible OpenCode agent orchestration.

## Expected behavior

- Be respectful and constructive.
- Assume good intent, but accept corrections quickly.
- Keep discussions technical and actionable.
- Prefer clear examples, reproduction steps, and concrete proposals.
- Respect maintainers' time and project scope.

## Unacceptable behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic comments or spam.
- Demanding support while ignoring documented guidance.

## Enforcement

Maintainers may edit, hide, or remove comments and issues that violate this code of conduct. Repeated or severe violations may lead to blocking from the project.

If you need to report a concern, open a private communication channel with the maintainer rather than posting sensitive details publicly.
